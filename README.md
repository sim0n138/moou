# План разработки игры «Змейка»

Этот репозиторий содержит проектную документацию для 2D-игры «Змейка» на Python с использованием Pygame. Основной документ со всеми подробностями находится в [`docs/snake_plan.md`](docs/snake_plan.md).

## Краткое содержание
- Цель проекта — подготовить кроссплатформенную версию игры и Windows-сборку `.exe`.
- Логика игры и визуализация разделены: отрисовка фиксирована на 60 FPS, перемещение змейки управляется отдельным таймером.
- Управление выполняется стрелками или WASD с буферизацией поворотов, исключающей мгновенный разворот.
- Игровое поле, рост змейки, генерация пищи и система очков опираются на дискретную сетку и детерминированные алгоритмы.
- Хранение рекордов совместимо с PyInstaller, ассеты располагаются в каталоге `assets/` и загружаются через вспомогательную функцию `resource_path`.

## Структура проекта
```
game/
├── __init__.py
├── config.py
├── resources.py
├── state.py
├── snake.py

assets/
├── fonts/

tests/
├── test_snake.py
├── test_food.py
└── test_score.py
```

## Следующие шаги
1. Настроить окружение (virtualenv, зависимости, линтеры, PyInstaller).
2. Реализовать модули конфигурации, ресурсов и базовую игровую модель.
3. Собрать основной игровой цикл с рендерингом, состояниями и пользовательским вводом.
4. Добавить подсчёт очков, хранение рекордов и аудио/визуальные улучшения.
5. Подготовить тесты, прогнать `pytest`, выполнить сборку через PyInstaller и описать процесс в README.

Подробный план и временная оценка доступны в [основном документе](docs/snake_plan.md).
